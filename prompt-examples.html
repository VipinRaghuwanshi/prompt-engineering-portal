Prompt Examples

Prompt examples help demonstrate how different types of instructions can be given to AI systems. A well-written prompt clearly explains the task and provides context to generate better results.

Education Prompt

Explain gravity to a 10-year-old child.

Programming Prompt

Write a Python function that reverses a string.

Creative Prompt

Write a motivational story about persistence.

Examples of Effective Prompts

Effective prompts are clear, detailed, and include context. Below are examples of prompts used for different tasks.

Educational Prompt

Prompt:

Explain the concept of photosynthesis in simple language for a middle school student.

Programming Prompt

Prompt:

Write a Python function that checks whether a number is prime or not.

Creative Writing Prompt

Prompt:

Write a short motivational story about perseverance and personal growth.

Business Prompt

Prompt:

Create a marketing strategy for launching a new mobile application.
